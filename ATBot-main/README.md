## ATBot主控功能如下
- 实时查看CPU和内存使用情况
- 记录群聊和私聊的聊天记录到log文件夹
- 计算接收和发送消息量（发送量只能计算当前插件内功能使用时的消息）
- 自定义帮助命令和菜单排版（默认/help）
- 通过发送/atinfo查询机器人的当前所有情况信息<br>
（CPU占用，内存占用，发送消息数量，接收消息数量，运行时长）
- 收到好友添加/群邀请会私聊给设置的主人QQ
